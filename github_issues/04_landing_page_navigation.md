# Implement a landing page and navigation

**As a** visitor
**I need** a landing page with navigation
**So that** I can understand GiftLink and navigate to other sections of the app

### Details and Assumptions
* Landing page displays the project title and tagline
* A "Get Started" button links to the main listings page
* Navigation bar provides links to Home, Search, Login, and Register

### Acceptance Criteria
```gherkin
Given I visit the GiftLink landing page
When the page loads
Then I see the site name, description, Get Started button, and navigation bar
```
